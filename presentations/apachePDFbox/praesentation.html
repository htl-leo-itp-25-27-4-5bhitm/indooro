Apache PDFBox

Was ist Apache PDFBox?

  • Eine Open-Source Java-Bibliothek zum Arbeiten mit PDF-Dateien
  • Unterstützt Lesen, Schreiben und Bearbeiten von PDFs
  • Teil der Apache Software Foundation

Was kann Apache PDFBox?

  • PDF-Text extrahieren
  • Bilder aus PDF extrahieren
  • Neue PDFs generieren
  • PDFs zusammenführen und teilen
  • Formulare bearbeiten

Vorteile

  • Java-basiert → leichte Integration in bestehende Anwendungen
  • Große Community & regelmäßige Updates
  • Kein kommerzielles Lizenzmodell notwendig
