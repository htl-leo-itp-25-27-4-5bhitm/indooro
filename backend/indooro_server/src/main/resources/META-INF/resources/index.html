Indooro LeoCloud

Wohin soll es gehen?

Diese Startseite fuehrt direkt zu den gehosteten Indooro-Werkzeugen in der LeoCloud. Der Admin-Editor speichert Layouts am Server. Die Kundensuche und die Swift-App koennen danach auf dasselbe aktuelle Layout zugreifen.

Admin Editor Layout bearbeiten, exportieren und direkt am Server speichern. Kundensuche Produkte suchen und das aktuell gespeicherte Layout im Browser pruefen. Backend Health Pruefen, ob die API im LeoCloud-Cluster erreichbar ist.
